How It Works

  1. Owner signs up at /owner/signup
  2. Owner creates tables in the dashboard
  3. Each table gets a unique QR code
  4. Customers scan QR to order from their table
  5. Orders appear in owner dashboard in real-time

Key Features

  • Owner authentication and session management
  • Dynamic QR code generation per table
  • Table-specific ordering experience
  • Order status tracking (pending/completed)
  • Customizable menu
